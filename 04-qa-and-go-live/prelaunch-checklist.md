## DNS & TLS

- Route 53 authoritative
- ACM certificate issued
- HTTPS enforced
- HTTP redirects working

---

## CloudFront

- Correct origin set
- TLS certificate attached
- Security headers enabled
- Canonical domain enforced

---

## Squarespace

- All pages published
- Mobile view reviewed
- Forms tested
- No placeholder content

---

## Content

- Homepage reviewed
- Architecture page accurate
- Services page scoped correctly
- About page complete

---

## Final Verification

- Test in incognito
- Test on mobile
- Verify no browser warnings
- Confirm performance acceptable

---

## Launch Rule

If something feels rushed, delay launch.
